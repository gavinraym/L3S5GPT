ChatGPT for Fun Coding

Introduction

The ChatGPT for Fun Coding class is all about learning how to use a cool tool called ChatGPT to create computer code that does what you want it to do! In this class, you'll learn how to tell ChatGPT what you want your code to do using everyday words, so you don't need to memorize tricky code or worry about getting the syntax right. On this track, you will create web sites!

By the end of the class, you'll be a pro at using ChatGPT to make your own code and you can use your new skills to make all sorts of cool things! This class is perfect for kids who are just starting to learn about computer programming, or for anyone who wants to have fun making code in a new and exciting way!

Class Expectations

This class is made up of a series of HTML files that include starter prompts demonstrating how ChatGPT can be leveraged to create web pages. As a student in this class, you are expected to run the starter prompt and add the resulting code to your project. After adding the code, you will find a commented section where you can explain what the generated code looks like and how you might change it.

For each file, you are expected to come up with five prompts that change your code in some way and record them in the file with an explaination of what it does. Each file has some example prompts that you can use as is or change to your liking. If tried and explained about, these prompts do count toward the 5 prompt requirment.

It's important to note that you do not have to keep the code generated from the prompts. You are free to go in new directions after you have recorded the prompts. The main goal of this class is to help you understand how to use ChatGPT to generate executable code and create web pages.

To succeed in this class, you should want to learn the basics of programming with HTML and be comfortable using a computer. You will also need to be able to follow instructions and complete assignments. If you have any questions or need extra help, please don't hesitate to ask your teacher. They are here to help you learn and succeed!

Our Hopes And Goals

Our hope for this class is that you become inspired to experiment with ChatGPT and use it to create unique web pages that reflect your interests and creativity. By working through the HTML files, you will have opportunities to start making these projects your own by taking them in different directions.

Our goal is to support you in taking a little extra time with each project, transforming it into a product of your creativity. The possibilities are endless, and we can't wait to see what you come up with!

Time To Get Started

Thank you for taking this class. We are sure you will enjoy learning about ChatGPT and how it can be used to create web pages. Remember, if you have any questions or need extra help, please don't hesitate to reach out to your teacher. They are here to support you and help you achieve your goals.
